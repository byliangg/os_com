# benchmark 日志目录

该目录是 ext4 benchmark 脚本的默认输出位置。

## 日志来源脚本

- `tools/ext4/run_phase4_in_docker.sh`
- `tools/ext4/run_phase4_part3.sh`
- `tools/ext4/run_phase4_part2.sh`
- `tools/ext4/run_phase4_part1.sh`
- `tools/ext4/run_phase3_dual_track.sh`

执行 `benchmark/sync_dataset.sh` 可将最新稳定结果同步到 `benchmark/datasets/results/`。
